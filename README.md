# Unfair Ordinance

Static advocacy website for UnfairOrdinance.org.

## What this site does

- Explains why the so-called FAIR Ordinance is weaker than the Protecting Renters Ordinance.
- Summarizes the core differences between the existing Chicago RLTO, PRO, and the counterproposal.
- Lets visitors draft a personalized email to their alderperson from their own email app.
- Stores no visitor data and sends no email server-side.

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. In Vercel, create a new project from that repository.
3. Use the default static settings. There is no build command.
4. Add `UnfairOrdinance.org` in Vercel under Project Settings -> Domains.
5. Point the domain's DNS records to Vercel when Vercel gives you the exact records.

## Publication maintenance

- Recheck the filed ordinance text and source links when the proposal changes.
- Recheck alderperson names, ward email addresses, and committee membership against the City Clerk directory before a new action alert.
- Update the review date shown in the page footer after refreshing public contact data.
- Keep the site disclaimer aligned with the campaign's current legal, lobbying, and organizational requirements.
